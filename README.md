# NavigatorMPEI
Навигатор для построения пути между кабинетами в НИУ МЭИ.  
Карта здания представляет собой svg-карту.  
Построение пути осуществляется с помощью алгоритам Ли.  

## Стек
С#, MS SQL, JS, Ajax, HTML, CSS, EntityFramework, MVC
